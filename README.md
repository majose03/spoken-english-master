# Spoken English Master

A beautifully designed, zero-backend, multi-lingual Spoken English Learning Web Application. Built entirely natively with HTML5, CSS3, and Vanilla JavaScript, ensuring lightening-fast loading times, offline persistent learning, and high accessibility.

## 🚀 Features

* **Zero Backend & No Frameworks**: 100% Client-side. No React, no Vue, no Node.js—just pure structural HTML, styling, and JS logic. Fast, simple, and fully portable.
* **22 Comprehensive Pages**: Includes Grammar Rules, Tenses, Modals, Phrasebooks, Phrasal Verbs, Quizzes, Pronunciation IPA, Essays, and more.
* **Native Multi-lingual Support**: Built specifically for South Indian learners, dynamically translating English context to Tamil, Malayalam, and Telugu natively without external APIs.
* **Web Speech API Integrations**: 
  * **Text-To-Speech (TTS)**: Built-in functionality allowing you to click icons and hear flawless native English pronunciations directly in your browser.
  * **Speech Recognition**: Practical Speaking Tasks leveraging the browser's microphone access to live-transcribe voice, grading fluency on the go.
* **Persistent Daily Tracking**: Uses the device `localStorage` to create offline habit trackers and streak markers without needing a database.
* **Premium UX/UI**: Dark mode glassmorphism interface with micro-glowing animations and CSS variables ensuring deep aesthetic engagement.

## 📂 Installation & Usage

No complex setup, build steps, or package managers required.

1. Clone or download the repository.
   ```bash
   git clone https://github.com/majose03/spoken-english-master.git
   ```
2. Open the directory.
3. Double-click on `index.html` to open it in any modern web browser (Google Chrome, Microsoft Edge, Safari, Firefox).
4. Jump straight into learning!
